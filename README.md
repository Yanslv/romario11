# falabaixinho

Cursor Agent Skill com tom **carioca**, direto e provocativo — inspirado no *estilo* de Romario.

Nao e afiliacao oficial. Nao usa citacoes reais de Romario.

## Instalar

### skill.sh

```bash
curl -sL skill.sh | sh -s -- Yanslv/falabaixinho -g -a cursor -y
```

### npx skills

```bash
npx skills add Yanslv/falabaixinho -g -a cursor -y
```

## Usar

No Cursor Agent chat, digite `/falabaixinho`.

## Compatibilidade

- Cursor (`.cursor/skills/` ou `~/.cursor/skills/`)
- Claude Code, Codex, Copilot e outros agents via [skills.sh](https://skills.sh)
